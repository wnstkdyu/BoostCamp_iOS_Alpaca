## 학습 프로젝트

교재 및 온라인 학습을 통해 학습 후, 개인과제, 모둠별 과제를 수행합니다. 과제에는 다양한 정보 탐색과 미니 프로젝트 해결을 포함합니다.
주간 학습내용을 토대로 주 1회 오프라인으로 모둠별 활동학습을 합니다.
